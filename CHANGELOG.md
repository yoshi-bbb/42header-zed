# Changelog

All notable changes to this project will be documented in this file.

The format follows Keep a Changelog, and this project uses semantic versioning.

## [0.1.0] - 2026-06-17

### Added

- Initial `zed-42header` CLI.
- Zed external formatter workflow for inserting and updating 42 headers.
- Support for slash, hash, semicolon, parenthesis, dash, and percent comment styles.
- Preservation of existing `Created` metadata when updating a header.
- Examples for Zed settings, keymap, and task usage.
